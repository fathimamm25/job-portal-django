A full-stack Job Portal web application built using Django.  
This platform connects **job seekers** and **employers**, allowing employers to post jobs and job seekers to apply with resumes.

---


## Job Seeker
- View available jobs
- Apply for jobs by uploading resume
  

## Employer
- Post new job openings
- View applicants for posted jobs

###  Authentication
- Custom user roles (Job Seeker / Employer)
- Secure login & registration

---

##  Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS
- **Database:** SQLite (local development)
- **Authentication:** Django Auth
- **Version Control:** Git & GitHub
